# Noveo Workout Tracker

A futuristic, mobile-first workout tracker built with HTML, CSS, and vanilla JavaScript.

## Features
- **Workout Tracking**: Log your sets, reps, and weights.
- **Custom Plans**: Create and manage your own workout routines.
- **History**: View your past workouts and track progress.
- **Profiles**: Support for multiple user profiles.
- **Offline Capable**: Works offline as a PWA.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- LocalStorage for persistence

## Setup
Simply open `index.html` in a modern browser.
To install as a PWA, open in Chrome/Safari on mobile and "Add to Home Screen".
